# ideathon
アイデアソンをみんなで回せば楽しいのではないかという思想の元、いろいろ策を考えてみようのコーナー


とりあえずいいと思ったのは、方法論的な話をした後に案だし、それに対するペルソナ設定などをこっちで決めて、どんなアイデアが出たのか楽しくはなしあえばいいのではの考えを持った。
これをする同期として楽しそうというのがあるが、新歓では楽しそうというおめめパキパキ人類だけでなく遊んでいたいノーマル人類もいるのでそれを取り込むためにその人たちがやる気になるようなアイデアづくりを考えたい.(技術力がなくてもできるという強みが、ハッカソンの弱体化みたいで簡単なのでやりたい感がある。)例えばどんなどうきづけが　できるだろうか

とりあえず課題を追加してくれる要素を追加してみた。
